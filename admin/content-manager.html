Quick Content Update Guide

  1. Identify the content you want to change
  2. Open the appropriate HTML file
  3. Find the text using Ctrl+F (Windows) or Cmd+F (Mac)
  4. Replace the old text with new content
  5. Save the file and test the changes

Content Guidelines

  • Keep content professional and aligned with club values
  • Maintain consistent tone and style across pages
  • Proofread all content before publishing
  • Keep descriptions concise but informative
  • Update copyright year in footer when needed
  • Ensure all links and contact information are current

Content Writing Tips

  • Write in active voice for better engagement
  • Use bullet points for easy scanning
  • Include specific achievements and metrics
  • Keep paragraphs short (2-3 sentences)
  • Use keywords relevant to communications and leadership
  • Test readability - content should be accessible to all audiences

How to Update Footer Information

The footer appears on every page and contains contact information:

  1. Footer content is in every HTML file
  2. Look for <footer class="main-footer">
  3. Update email, address, and social media links
  4. Remember to update ALL HTML files for consistency

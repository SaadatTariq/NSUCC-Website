How to Manage Gallery Images

Step 1: Prepare Your Images

  1. Use high-quality images (minimum 1920x1080 pixels for best results)
  2. Save images as JPG or PNG format
  3. Optimize file sizes (keep under 2MB each for faster loading)
  4. Name files descriptively (e.g., "event2024_1.jpg", "workshop_photo.jpg")
  5. Upload images to: assets/images/gallery/

Step 2: Update Gallery Slider (Homepage)

The homepage features a photo slider. To update these images:

  1. Open index.html
  2. Find the section with <div class="gallery-slider">
  3. Replace image file names in the slider
  4. Ensure slide navigation numbers match the number of images

Step 3: Quick Image Replacement

To replace images without changing the website structure:

  1. Keep the same file name as the existing image
  2. Replace the old image file with your new image
  3. Clear browser cache (Ctrl+F5 or Cmd+Shift+R) to see changes

Image Replacement Note

This method is easiest but requires your new image to have the exact same filename as the old one.

Step 4: Add Background Images

Update background images used throughout the site:

  1. Main background: bg.png
  2. Gallery title image: Title.JPEG
  3. Other decorative images: photo4.jpg

Image Guidelines

  • Always backup your gallery folder before making changes
  • Optimize images for web to ensure fast loading
  • Maintain consistent image quality and style
  • Test gallery functionality after adding new images
  • Ensure images are appropriate and represent the club well
  • Consider image orientation (landscape works best for sliders)

Image Optimization Tips

  • Use JPG for photographs, PNG for graphics with transparency
  • Resize images to appropriate dimensions before uploading
  • Compress images using online tools (TinyPNG, etc.)
  • Maintain aspect ratios to prevent distortion
  • Use descriptive alt text for accessibility
  • Keep a backup of original high-resolution images

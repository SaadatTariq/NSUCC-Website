Logo Update Instructions

To update logos:

  1. Replace the logo files in assets/images/
  2. Keep the same filenames for automatic updates
  3. Use PNG format with transparent background
  4. Recommended size: 200x200 pixels for logo.png

Important Notes

  • Always backup your files before making changes
  • Settings updates need to be applied to multiple HTML files
  • Test all social media links after updating
  • Update copyright year annually
  • Keep contact information current and accurate
  • Ensure social media accounts are active and monitored

Quick Settings Checklist

At the beginning of each term, update:

  • ☐ Term year in executive page header
  • ☐ Contact email if changed
  • ☐ Social media links if accounts changed
  • ☐ Room number if club moved
  • ☐ Copyright year in footer
  • ☐ Current session information

Global Find & Replace Guide

To update settings across all files quickly:

  1. Use a text editor that supports "Find in Files" (VS Code, Notepad++, etc.)
  2. Open the entire website folder
  3. Use Find & Replace to update all instances at once
  4. Always preview changes before applying
  5. Test the website after making changes

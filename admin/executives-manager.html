How to Update Executive Team

Step 1: Prepare Executive Photos

  1. Take or obtain high-quality photos of executives (preferably 800x1200 pixels)
  2. Save photos as PNG or JPG format
  3. Name files descriptively (e.g., "President.png", "Vice-President.png")
  4. Upload photos to: assets/images/executives/

Step 2: Update Executive Information

  1. Open the file: executives.html
  2. Find the executive card you want to update
  3. Replace the photo path, name, title, and description
  4. Update social media links if needed

Step 3: Quick Photo Replacement Guide

To quickly replace photos without changing text:

  1. Keep the same file name as the current photo
  2. Replace the old photo file with the new one
  3. Clear your browser cache (Ctrl+F5 or Cmd+Shift+R) to see changes

Important Notes

  • Always backup your executives.html file before making changes
  • Keep photo file sizes reasonable (under 2MB each)
  • Maintain consistent photo dimensions for better layout
  • Test the page after updates to ensure proper display
  • Update social media links to current profiles

Pro Tips

  • Use consistent background colors for executive photos
  • Write bios in third person for professionalism
  • Keep descriptions between 100-200 words
  • Include achievements and leadership qualities
  • Verify all social media links are working

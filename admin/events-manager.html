How to Add a New Event

Step 1: Prepare Your Event Logo

  1. Create or obtain your event logo (PNG or JPG format recommended)
  2. Resize the image to approximately 300x300 pixels for best results
  3. Name the file descriptively (e.g., "WordCraft2025.png")
  4. Upload the logo to: assets/images/event_logo/

Step 2: Edit the Events Page

  1. Open the file: events.html
  2. Find the section that says <!-- Add new events here -->
  3. Copy the template below and paste it after the last event card
  4. Replace the placeholder information with your event details

Important Tips

  • Always backup your events.html file before making changes
  • Make sure your logo file is uploaded before adding the event
  • Test the page after adding to ensure everything displays correctly
  • Keep descriptions concise but informative
  • Use consistent formatting for better visual appeal

Need Help?

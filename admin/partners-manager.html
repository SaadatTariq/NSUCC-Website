How to Manage Partners

Step 1: Prepare Partner Logos

  1. Obtain high-quality partner logos (PNG, JPG, or SVG format)
  2. Resize logos to approximately 200x200 pixels (square format works best)
  3. Use transparent backgrounds for PNG files when possible
  4. Name files descriptively (e.g., "CompanyName.png")
  5. Upload logos to: assets/images/partners/

Step 2: Add New Partner

  1. Open the file: partners.html
  2. Find the partners grid section
  3. Copy an existing partner card template
  4. Replace with new partner information
  5. Update the partner count in statistics if needed

Step 3: Remove a Partner

  1. Open partners.html
  2. Find the partner card you want to remove
  3. Delete the entire <div class="partner-card">...</div> section
  4. Update the partner count in the statistics section
  5. Optionally, delete the logo file from the partners folder

Step 4: Update Partner Statistics

After adding or removing partners, update the count in the statistics section:

  1. Find the section with <span class="stat-number">25+</span>
  2. Update the number to reflect current partner count
  3. If you have significantly more partners, consider updating other stats too

Important Guidelines

  • Always backup partners.html before making changes
  • Ensure logo files are optimized for web (under 500KB each)
  • Test the page after adding partners to ensure proper display
  • Maintain consistent logo quality and style
  • Get permission before adding partner logos
  • Keep partner logos in a professional, clean format

Pro Tips for Partner Management

  • Group similar partners together (e.g., all banks, all tech companies)
  • Use consistent animation delays (increment by 25-50ms)
  • Consider partner hierarchy when ordering (major partners first)
  • Regularly review and remove inactive partnerships
  • Update statistics to reflect current partnership impact

## ğŸ–¼ï¸ Image Upload System

### How to Upload Images
1. **Open Admin Panel**: Go to the Image Upload Center
2. **Choose Category**: Select from Executive Photos, Event Logos, Partner Logos, or Gallery Images
3. **Upload Method**: 
   - **Drag & Drop**: Simply drag images from your computer to the upload area
   - **Click to Browse**: Click the upload area and select images from your computer
4. **Wait for Upload**: Watch the progress bar as images upload
5. **Confirm**: You'll see a preview of uploaded images

### Image Guidelines
- **File Formats**: JPG, PNG, WebP, SVG
- **File Size**: Maximum 5MB per image
- **Executive Photos**: 500x600px recommended
- **Event Logos**: Square format preferred
- **Partner Logos**: Transparent background preferred
- **Gallery Images**: High resolution for best quality

## ğŸ‘¥ Executive Team Management

### Adding New Executive Members
1. Go to **Executive Team** tab in admin panel
2. Click **"Add New Executive"**
3. Fill out the form:
   - Upload photo (professional headshot recommended)
   - Enter name
   - Enter position title
   - Write description/bio
   - Add social media links (optional)
4. Click **Save**

### Updating Existing Executives
1. Find the executive card you want to update
2. Click **"Edit"** button
3. Modify any information needed
4. Upload new photo if needed
5. Click **Save**

### Changing Photos for New Terms
1. Go to **Image Upload Center**
2. Upload new executive photos to **Executive Team Photos** section
3. Go to **Executive Team** tab
4. Edit each executive and replace their photo
5. Update names, positions, and descriptions as needed

## ğŸ“… Event Management

### Adding New Events
1. Go to **Events Manager** tab
2. Click **"Add New Event"**
3. Upload event logo
4. Enter event name
5. Write event description
6. Add event date
7. Set event status (Completed/Upcoming/Ongoing)
8. Click **Save**

### Updating Event Information
1. Find the event you want to update
2. Click **"Edit"** button
3. Modify information or upload new logo
4. Click **Save**

## ğŸ¤ Partner Management

### Adding New Partners
1. Go to **Image Upload Center**
2. Upload partner logos to **Partner Logos** section
3. Go to **Partners Manager** tab
4. The new logos will appear in the partners grid
5. Partners are automatically added to the website

### Removing Partners
1. Go to **Partners Manager** tab
2. Find the partner you want to remove
3. Click the delete button (trash icon)
4. Confirm removal

## ğŸ–¼ï¸ Gallery Management

### Adding Gallery Photos

## âš ï¸ Important Safety Tips

### Before Making Changes
1. **Backup First**: Always backup your files before making major changes
2. **Test Locally**: If possible, test changes on a copy first
3. **Start Small**: Make one change at a time when learning

### Image Best Practices
1. **Optimize Images**: Use appropriate file sizes (not too large)
2. **Consistent Naming**: Use simple file names without spaces
3. **Quality Check**: Ensure images are clear and professional

### When Things Go Wrong
1. **Don't Panic**: Most issues can be fixed
2. **Restore Backup**: Use your backup files if needed
3. **Ask for Help**: Contact your website administrator
4. **Document Issues**: Note what you were doing when problems occurred
